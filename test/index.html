App -> App: 获取当前登录的 UserId App -> App Server: 通过 UserId 请求 Token App Server -> Rong Cloud API Server: 通过 UserId 请求 Token Rong Cloud API Server --> App Server: 返回 Token App Server -> App Server: 数据库存储 Token App Server --> App: 返回 Token App --> App: 返回 Token App -> App: 本地存储 App -> Rong Cloud IM Server: 通过 connect 方法附带 Token 连接到聊天服务器
